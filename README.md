TEAM14 저가 화장품 트렌드 소비자 반응 분석 (다이소 뷰티)

<팀원>
경영학과 202643251 이시연, 경영학과 202643328 허정윤


<프로젝트 소개>
최근 MZ세대 사이에서 빠르게 유행하고 있는 다이소 뷰티 제품의 리뷰를 분석하여, 과연 소비자에게 "가격"뿐 아니라 "품질"까지도 인정받고 있는지 직접 검증하는 것을 목표로 한다. 

다이소몰 뷰티/위생 카테고리 중 스킨케어, 메이크업, 뷰티제품 총 3가지 카테고리를 수집 대상으로 선정하였다. 그 중 '리뷰많은순' 설정을 통해 상위 3개의 제품을 집중적으로 분석한다.


<수집 데이터: 다이소몰 뷰티/위생 카테고리 中>
1. 스킨케어 카테고리, 리뷰 많은 순
- VT 리들샷 100 페이셜 부스팅 퍼스트 앰플 2ml
- 마데카21 데카 솔루션 수딩 미스트 토너 200ml
- 본셉 비타씨 동결 건조 더블샷 앰플 키트

2. 메이크업 카테고리, 리뷰 많은 순
- 입큰 퍼스널 톤 코렉팅 블러 팩트 5.5g(라벤더)
- 클라뷰 밸런싱 포어밤 프라이머 10ml
- 태그 슬림브로우펜슬(2호_애쉬브라운)

3. 뷰티제품 카테고리, 리뷰 많은 순
- [보송매끈]밀착쿠션에어퍼프(2p)
- 더마퀸 쿨링스틱
- [촉촉윤광]퍼피 촉촉쿠션 통퍼프


<실행 방법>
- 1단계: 라이브러리 설치
pip install selenium beautifulsoup4 pandas openpyxl
pip install matplotlib wordcloud streamlit webdriver-manager

- 2단계: crawl.py 를 실행해 크롤링 및 분석 (약 6분 소요)

- 3단계: dashboard.py 를 실행해(터미널에 streamlit run dashboard.py 입력) 시각화 확인


<사용 라이브러리>
1. crawl.py
- selenium — 다이소몰 페이지 자동 조작 및 리뷰 크롤링
- beautifulSoup (bs4) — 수집한 HTML에서 리뷰 텍스트, 별점, 날짜 파싱
- pandas — 수집 데이터 정리 및 CSV/엑셀 저장
- webdriver-manager — 크롬 드라이버 자동 설치 관리

2. dashboard.py
- streamlit — 웹 대시보드 시각화
- matplotlib — 막대그래프 시각화
- wordcloud — 키워드 워드클라우드 생성
- pandas — CSV 데이터 로드 및 가공


<파일 설명>
- crawl.py: 다이소몰 뷰티 제품 리뷰를 자동으로 수집하고 감성분석 후 CSV/엑셀 저장
- dashboard.py: crawl.py가 저장한 CSV를 불러와 streamlit 웹 대시보드로 시각화


<구현 결과>
1. crawl.py
- 제품당 최대 100개 리뷰 수집, 전처리(빈 리뷰·중복 제거) 후 분석
- 각 리뷰에 대해 긍정/부정/중립 감성 분류 및 속성별(가격, 효과, 텍스처, 재구매의사, 패키징) 키워드 언급 여부 분석
- 결과를 CSV, 엑셀(시트 3개) 형태로 저장

2. dashboard.py
- 상단 요약 지표: 전체 리뷰 수, 평균 별점, 긍정 비율, 부정 비율
- 카테고리별 평균 별점 막대그래프
- 카테고리별 긍정/부정/중립 감성 비율 막대그래프
- 제품별 평균 별점 비교 막대그래프 (9개 제품 한눈에 비교)
- 속성별 키워드 언급 비율 막대그래프 (가격 vs 품질 핵심 질문 답변)
- 카테고리별 긍정/부정 키워드 워드클라우드 총 6개 (긍정=초록, 부정=빨강)
- 수집된 전체 리뷰 데이터 테이블
